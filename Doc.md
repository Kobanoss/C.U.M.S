# DOCUMENTATION FOR C.U.M.S.


### BASE OPERATIONS LIST

#### OP_PUSH_INT (push value)
Desc: 
	Adding integer value to top of the stack

Arguments:
/n/tInteger value
Input: 
	None
Output: 
	None

Example:
	push 5


#### OP_DROP (drop) 
Desc:
	Removing integer value from top of the stack

Arguments:
	None
Input:
	None
Output:
	None
